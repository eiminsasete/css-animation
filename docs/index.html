transition

trasitionプロパティとは、時間的変化をまとめて指定するためのもので、以下の4つのプロパティを含んでいます。

表：transitionの内包するプロパティ
プロパティ 値
transition-property

時間的変化を適用するCSSプロパティを指定します。

表：transition-propzertyプロパティに指定できる値
値
all 全てのプロパティ（初期値）
none どのプロパティも変化しない
プロパティ名 記述したプロパティ。
カンマ区切りで複数指定可能
transition-duration

変化に掛かる時間を指定する

値は初期値が0

単位はs（秒）またはms（ミリ秒）
transition-timing-function

変化のタイミングを指定します。

表：transition-timing-functionに指定できる値
値
ease 最初と最後が滑らかに変化（初期値）
linear 一定に変化
ease-in 最初のみゆっくり
ease-out 最後のみゆっくり
ease-in-out 最初と最後でゆっくり
cubic-bezier(i, i, i, i) 3次ベジェ曲線で変化を指定する。iは0.0から1.0の正の値。
transition-delay

変化の始まるタイミングを指定します。

値は初期値が0

単位はs（秒）またはms（ミリ秒）

負の値を指定すると、時間をさかのぼって即変化が始まる。

transform

transformプロパティは、要素を二次元または三次元的に変形するためのもので、回転、拡大・縮小、平行移動、剪断及びこれらを内包するマトリクス変形、遠近効果を適用できます。

transformプロパティでは値にnone（初期値）またはtransform関数を用います。またホワイトスペースで区切って複数指定することも可能です。

表：transform関数一覧表
transform関数
translate系
  • translateX(dX)
  • translateY(dY)
  • translateZ(dZ)
  • translate(dX,dY)
  • translate3d(dX,dY,dZ)

K=X,Y,Zとして引数dKはK軸への移動距離です。

transform関数のtranslate系はK軸方向にdKだけ要素を平行移動させます。

引数の単位は長さ系（px,em,remなど）とパーセント(%)です。
ただしZ軸ではパーセントは無効です。

Z軸方向への操作にはtransform関数perspectiveの併用が必須です。
scale系
  • scaleX(pX)
  • scaleY(pY)
  • scaleZ(pZ)
  • scale(pX,pY)
  • scale3d(pX,pY,pZ)

K=X,Y,Zとして引数pKはK軸方向の倍率です。

transform関数のscale系はK軸方向に要素をpK倍にします。

引数の単位はありません。
rotate系
  • rotateX(aX)
  • rotateY(aY)
  • rotateZ(aZ)
  • rotate(aX,aY)
  • rotate3d(aX,aY,aZ)

K=X,Y,Zとして引数aKはK軸を軸とした回転する角度。

transform関数のrotate系はK軸を軸として要素をaK度回転させます。

引数の単位は角度(deg)です。
skew系
  • skewX(aX)
  • skewY(aY)
  • skew(aX,aY)

K=X,Y,Zとして引数aKはK軸に沿って傾く角度。

transform関数のskew系はK軸に沿って要素をaK度傾けさせます。

引数の単位は角度(deg)です。
perspective系
  • skewX(aX)
  • skewY(aY)
  • skew(aX,aY)

K=X,Y,Zとして引数aKはK軸に沿って傾く角度。

transform関数のskew系はK軸に沿って要素をaK度傾けさせます。

引数の単位は角度(deg)です。

animation

animationプロパティは、HTML要素にキーフレームアニメーションを適用し、細かい動きのアニメーションを実装することができます。

animationプロパティでは、animation-name、 animation-duration、 animation-timing-function、 animation-delay、 animation-iteration-count、 animation-directionなどのプロパティの値をスペースで区切ることで複数指定できます。

キーフレームアニメーションとは

キーフレームアニメーションとはアニメーションの開始(0%)から終了(100%)までの任意の経過地点(キーフレーム)に プロパティを指定できるアニメーションのこと。

表：animationプロパティの包括するプロパティ一覧
プロパティ
animation-name HTML要素に適用するキーフレームアニメーションの名前を指定します。 
.sample-1{animation-name: expandWidth;}
.sample-2{animation-name: beTransparent;}
@keyframes expandWidth{
  0%{width: 10%;}
  50%{width: 20%;}
  100%{width: 100%;}
}
@keyframes beTransparent{
  0%{opacity: 1;}
  50%{opacity: .5;}
  100%{opacity: 0;}
}
animation-duration キーフレームアニメーション一回分(0%から100%まで)の時間の長さを指定します。 
.sample-1{
  animation-name: expandWidth;
  animation-duration: 3s;
}
@keyframes expandWidth{
  0%{width: 10%;}
  50%{width: 20%;}
  100%{width: 100%;}
}
animation-timing-function キーフレームアニメーションのタイミング・進行割合を指定します。
animation-timing-functionの値
ease
  • 初期値
  • cubic-bezier(0.25, 0.1, 0.25, 1.0)と同じ
  • 進行割合の開始と完了を滑らかにします。
linear
  • cubic-bezier(0.0, 0.0, 1.0, 1.0)と同じ
  • 進行割合を一定にします。
ease-in
  • cubic-bezier(0.42, 0, 1.0, 1.0)と同じ
  • 進行割合の開始を遅くします。
ease-out
  • cubic-bezier(0, 0, 0.58, 1.0)と同じ
  • 進行割合の完了を遅くします。
ease-in-out
  • cubic-bezier(0.42, 0, 0.58, 1.0)と同じ
  • 進行割合の開始を完了を遅くします。
cubic-bezier(x1, x2, y1, y2)
  • 3次ベジェ曲線のP1とP2を (x1, y1, x2, y2) で指定します。
 
.sample-1{
  animation-name: expandWidth;
  animation-duration: 3s;
  animation-timing-function: ease;
}
@keyframes expandWidth{
  0%{width: 10%;}
  50%{width: 20%;}
  100%{width: 100%;}
}
animation-delay キーフレームアニメーションがいつ始まるかを指定します。 
.sample-1{
  animation-name: expandWidth;
  animation-duration: 3s;
  animation-timing-function: ease;
}
.sample-1.type-1{
  /* アニメーションは1秒後に実行されます。 */
  animation-delay: 1s;
}
.sample-1.type-2{
  /* アニメーションは1秒経った経過地点から即実行されます。 */
  animation-delay: -1s;
}
@keyframes expandWidth{
  0%{width: 10%;}
  50%{width: 20%;}
  100%{width: 100%;}
}
animation-iteration-count

キーフレームアニメーションの繰り返し回数を指定します。

指定できる値 効果
infinite 無限に再生を繰り返します
数値(自然数) 数値と同じ回数、再生されます 
.sample-1{
  animation-name: expandWidth;
  animation-duration: 3s;
  animation-timing-function: ease;
  animation-delay: 1s;
  animation-iteration-count: infinite;
}
.sample-1.test{
  animation-iteration-count: 1;  
}
@keyframes expandWidth{
  0%{width: 10%;}
  50%{width: 20%;}
  100%{width: 100%;}
}
animation-direction

キーフレームアニメーションを交互に反転再生させるかどうかを指定します

指定できる値 効果
normal

初期値

普通方向の再生でアニメーションサイクルを繰り返します
alternate

奇数回では普通方向の再生、偶数回では逆方向の再生となって、アニメーションサイクルを繰り返します

 
.sample-1{
  animation-name: expandWidth;
  animation-duration: 3s;
  animation-timing-function: ease;
  animation-delay: 1s;
  animation-direction:normal;
}
.sample-1.reverse{
  animation-direction:alternate;
}
@keyframes expandWidth{
  0%{width: 10%;}
  50%{width: 20%;}
  100%{width: 100%;}
}
